Save As

Location

Browser local storage

Save the map to the browser local storage area. Useful for quick edits, but limited in size, and can be more easily lost.

Zip Export

Save the map to a zip file.

STL Export

Export the heightmap in STL format for 3D printing. You can use slicing software like Slic3r, Cura or Simplify3d to convert an STL file to a format that can be printed.

Base thickness

    This is additional thickness to add below the lowest point on the map, expressed in mm. It assumes the longest edge of the print is 10cm long.

    Vertical scale

      Heightmaps printed can appear excessively flat. Setting this value above 100% exagerates the vertical height of the map without affecting the base thickness.

      Raw Heightmap Export

      Save the heightmap only to a Raw file. This can be edited in an external Application for finer control, then re-imported from the open dialog.
